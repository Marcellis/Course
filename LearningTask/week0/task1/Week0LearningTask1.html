Learning outcomes

Students learn how to build native Android apps:
  • Novice Level:
    • How to build user interfaces
    • How to apply adapters
    • How to implement SQLite
    • How to use the recycler view
  • Professional level:
    • How to apply fragments
    • How to implement the content provider
  • Expert level:
    • How to use google Maps

Prerequisites

To be prepared for learning Android you have to know the basics of Java, Object oriented programming and XML. You can use the button support to follow Lynda.com courses to upgrade your knowledge.
